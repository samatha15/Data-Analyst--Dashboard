# Data-Analyst--Dashboard
Interactive Excel dashboard analyzing music streaming habits using Pivot Tables, Charts, KPI Cards, and Slicers to visualize user engagement, listening time, skip rates, and platform performance.

#  Music Streaming Habits Dashboard

##  Project Overview
This project analyzes music streaming behavior using Microsoft Excel. The dashboard provides insights into user listening patterns, average listening time, age distribution, skip rates, and platform usage.

##  Objectives
- Analyze user listening habits.
- Compare listening time across countries.
- Identify average age of listeners.
- Analyze skip rates by platform.
- Compare subscriptions and songs played.

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

##  Dashboard Metrics
- Total Users: 4000
- Average Listening Time: 138.70 mins
- Average Age: 27.85 years
- Average Skip Rate: 28.16%
- Total Songs Played: 156,550

##  Visualizations
1. Average Listening Time by Country
2. Average Age by Country
3. Average Skip Rate by Platform
4. Average Listening Time by Platform
5. Subscription vs Songs Per Day by Country

##  Key Insights
- Listening time varies significantly across countries.
- Different platforms show different skip rate patterns.
- Average listener age is approximately 28 years.
- Spotify and YouTube Music have strong user engagement.
- Total songs played exceed 150,000.
